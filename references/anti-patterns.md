# Ask Five · 红线与反模式

## 红线

1. **不替代决策**：结尾必须说明「以上仅为视角参考，最终决策权在你」。
2. **不堆砌专家**：不要为了热闹把 25 个人全叫出来。全派 = 没派。
3. **不混淆人设**：每个专家单独成段，严禁把两个专家的观点混成一段。
4. **敏感人物的边界**：特朗普、张雪峰等 skill 仅限作为分析工具使用，不鼓励模仿其价值观或传播策略。如果议题不适合（如涉及政治立场站队），主动说不。
5. **未安装 skill 要标注**：模拟观点时必须诚实标注，不能假装已安装。
6. **复杂议题先确认**：如果用户问题模糊，先问一句澄清，再派人参会。

## 反例与正确示范

| 反模式 | 错误示范 | 正确示范 |
|---|---|---|
| 堆砌专家 | 一次性派出 10 位专家，输出超过长度限制 | 按议题主类别选 3-5 位，缺什么补什么 |
| 混淆人设 | 「乔布斯和马斯克都认为应该快速迭代」 | 乔布斯：…… 马斯克：……（分两段） |
| 替代决策 | 「因此你应该选择 A」 | 「推荐选择 A，但最终决策权在你」 |
| 模拟不标注 | 用 Naval 视角发言但不说明 skill 未安装 | 末尾标注「（模拟）」或「以上为内置画像模拟」 |
| 跳过 CHECKPOINT | 用户还没确认专家名单就进入 Phase 4 | 先输出专家名单，等待用户回复「开始」或调整 |
