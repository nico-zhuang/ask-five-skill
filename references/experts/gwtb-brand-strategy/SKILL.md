# GWTB 品牌沟通策略分析

> GWTB 是 BBDO 的核心策略工具，用四个问题定义品牌的沟通逻辑。

## 触发场景

- 用户扔过来一份品牌笔记/评论数据，问「这些品牌都在讲什么」
- 用户需要竞品沟通策略对比
- 议题涉及品牌信息地图、差异化定位、传播策略诊断

## GWTB 四要素

| 要素 | 回答的问题 | 能从什么数据中看出 |
|---|---|---|
| **GET** | 品牌在对谁说话？ | 达人标签、内容主题、评论人群特征 |
| **WHO** | 消费者现在怎么想/有什么障碍？ | 消费者评论（最直接） |
| **TO** | 品牌希望他们做什么/信什么？ | 品牌笔记的 CTA、价值主张 |
| **BY** | 品牌用什么大创意来实现？ | 品牌笔记的标题风格、内容母题、语言模式 |

## 核心原则

1. **必须先做高频词提取** — 词是策略的原材料
2. **GWTB 是定性分析，不是打分** — 四个要素各自是策略判断，不是数字
3. **不确定就说不确定** — WHO 看不出来就标 🔴 数据不足
4. **数据源决定分析深度** — 品牌笔记看 GET + BY；消费者评论看 WHO
5. **Tag 切割是可选的，不是强制的** — 没有 Tag 字段也能跑完整分析

## 工作流

### Phase 0：数据识别与 Tag 可用性检查

#### 0.1 自动识别数据类型

| 数据特征 | 数据类型 | 能支撑什么 |
|---|---|---|
| 含「标题/title」「笔记内容」「正文」 | 笔记/帖子内容 | GET + BY 判断 |
| 含「评论/comment」，文本短、口语化、含情绪词 | 消费者评论 | WHO 的核心数据 |
| 含「搜索词」「上游」「下游」 | 搜索行为数据 | GET 验证、品类信息结构 |
| 含竞品品牌名 + 同类字段 | 竞品数据 | 竞品 GWTB 对比 |

#### 0.2 Tag 可用性检查

检查数据中是否包含可用于品牌 Tag 切割的字段：
- 是否有「话题标签/tags/标签」列？
- 标题或正文中是否出现 `#品牌名` 格式？
- 是否有「是否商业笔记/报备合作品牌」等字段？

如果可以切割，向用户说明两条路线：
- **路线 A（品牌投放）**：带 #品牌Tag 的笔记，反映品牌想让你看到的东西
- **路线 B（野生讨论）**：不带 Tag 但提及品牌的笔记，反映路人真实讨论

如果数据不能切，直接跳过，标注「数据不含 Tag 信息，以全量数据分析」。

#### 0.3 数据源质量判定

| 数据源类型 | 能可靠判断的要素 | 不能判断的 |
|---|---|---|
| 品牌官方号/达人笔记（路线 A） | GET、BY | WHO |
| 野生 UGC（路线 B） | WHO | GET、BY |
| 消费者评论 | WHO | — |
| 品牌笔记 + 消费者评论都齐全 | GET、WHO、TO、BY 全四要素 | — |

### Phase 1：高频词提取（必须执行）

- 对每个品牌和消费者评论分别提取：
  1. **内容母题**：品牌在讲什么场景？
  2. **品牌卖点语言**：品牌用什么词形容自己？
  3. **消费者实际关注点**：评论中高频出现什么？
  4. **语言模式 GAP**：品牌说的 vs 消费者说的差异

- 输出校准结果：
```
=== 品牌 A 高频词（标题端）===
复古(38) 穿搭(16) 日系(10) ...
→ BY 方向：日系生活方式 + 运动场景

=== 消费者评论高频词 ===
舒服(134) 好穿(151) 丑(27) 滑(47) ...
→ WHO 方向：功能体验为核心关注，外观争议明显

=== GAP ===
品牌在讲「生活方式美学」，消费者在讨论「功能体验细节」
```

### Phase 2：逐品牌 GWTB 拆解

如果做了 Tag 切割，对路线 A 和路线 B 分别跑 GWTB：
- **路线 A GWTB**：品牌想让你相信的 GWTB
- **路线 B GWTB**：路人眼中品牌实际的 GWTB

如果没做切割，用全量数据跑一套 GWTB。

#### GET — 在对谁说话？

```
GET: [一句话目标受众描述] 🟢/🟡/🔴
证据：[数据中的具体线索]
不确定时：🔴 "数据未明确指向特定人群，可能是泛人群沟通"
```

#### WHO — 消费者现在怎么想？有什么障碍？

```
WHO: [当前认知/感受/障碍] 🟢/🟡/🔴
证据：[消费者评论原声引用]
不确定时：🔴 "无消费者评论数据，WHO 无法可靠判断"
```

#### TO — 希望他们做什么/信什么？

```
TO: [期望的行为改变或认知重构] 🟢/🟡/🔴
证据：[品牌 CTA 或价值主张线索]
不确定时：🔴 "品牌内容缺乏明确的行动引导，TO 模糊"
```

#### BY — 用什么大创意来实现？

```
BY: [一句话核心命题] 🟢/🟡/🔴
证据：[高频词模式 + 代表性标题]
不确定时：🔴 "品牌内容调性分散，未能识别出统一的大创意"
```

#### 路线 A vs B 对比（仅当做了 Tag 切割）

- 如果 A 和 B 撕裂严重 → 品牌沟通策略存在方向性偏差
- 如果 A 和 B 一致 → 品牌沟通策略落地有效

### Phase 3：跨品牌对比分析

1. **GWTB 策略总览表**：品牌 × GET × WHO × TO × BY × 置信度
2. **信息重叠检测**：找出「大家都在讲一样的东西」
3. **信息空白**：找出「没人讲但消费者在关心的东西」
4. **品牌沟通一致性检验**：品牌标题高频词 vs 消费者评论高频词 → GAP 分析

## 重要提醒

1. **Tag 切割不是目的，是手段**
   - 没有 Tag 字段也能跑完整 GWTB 分析
   - 不要因为缺少 Tag 而告诉用户「数据不够」

2. **GWTB 不是填空题**
   - WHO 没数据就写 🔴 无法判断
   - 一个诚实的空白比一个编造的判断有价值

3. **BY 的判断标准**
   - 好的 BY：一句话能说清、创意人员看完有画面、不是产品功能罗列
   - 如果品牌的 BY 是三句话五个卖点，如实记录「BY 不清晰」

4. **品类上下文**
   - 如果所有品牌都在讲「专业科技」，那它只是入场券而非差异化
   - 报告中标注每个 BY 在品类中是「入场券」还是「差异化」
